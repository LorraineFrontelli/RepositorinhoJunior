# Repositorinho Junior
Por meio da atividade de DevOps do professor Carlos Alberto (querido Carlinhos), desenvolvemos uma página pessoal para cada integrante do grupo explicando e abrangendo lugares que o grupo tem vontade de conhecer.

### Resumo da atividade
A atividade tinha o objetivo de práticar revisões de PRs no GitHub por meio da interface gráfica, onde cada membro era responsável pela PR de um outro membro exclusivamente. Portanto, as PRs deveriam ser significativas e deveriam envolver um projeto de site pessoal envolvendo o lugar dos sonhos de cada membro.

## Sites pessoais

### Eduardo Rodrigues
