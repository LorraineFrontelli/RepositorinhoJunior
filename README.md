# Lugar dos Sonhos

Projeto escolar colaborativo onde cada integrante do grupo criou uma página pessoal apresentando o **lugar dos seus sonhos**.  
A tela inicial (`index.html`) centraliza todos os colaboradores, exibindo uma foto de cada um. Ao clicar na foto, o visitante é direcionado para o site individual do aluno, onde encontra informações sobre o lugar escolhido, curiosidades e motivos para visitá-lo.

## Como começar

Este projeto não requer instalação de dependências.  
Para visualizar:

1. Baixe ou clone o repositório.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Clique nas fotos dos colaboradores para acessar os sites individuais.

## Resumo da atividade
A atividade tinha o objetivo de práticar revisões de PRs no GitHub por meio da interface gráfica, onde cada membro era responsável pela PR de um outro membro exclusivamente. Portanto, as PRs deveriam ser significativas e deveriam envolver um projeto de site pessoal envolvendo o lugar dos sonhos de cada membro.

## Funcionalidades

- Tela inicial com fotos e nomes dos colaboradores.
- Links para páginas individuais de cada aluno.
- Cada página apresenta:
  - Informações sobre o aluno.
  - Descrição do lugar dos sonhos.
  - Imagem ilustrativa.
  - Curiosidades sobre o local.

## Tecnologias utilizadas

- **HTML5** para estrutura das páginas.  
- **CSS3** para estilização e design responsivo.
